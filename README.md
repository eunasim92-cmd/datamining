## 주요 발견

- 역 거리(p=0.04)는 유의미하나 실질 차이 27m에 불과, 유동인구(p=0.066)는 유의미하지 않음
- 두 변수 조합해도 통계적으로 유의미하지 않음 (p=0.34) — 접근성 변수들이 폐업과 일정한 관련 가능성은 있지만, 조합변수의 효과가 명확하게 나타나지 않아서 단일 변수로 해석하기에는 제한이 있음
- 시설면적과 폐업의 상관관계가 가장 뚜렷(p≈0) — 단, 인허가 시기 교란 가능성 있음
- 2007 ~ 2009, 2016 ~ 2017, 2024 ~ 현재 세 차례 개업 급증 주기 관찰
